# COMP9517-computer-vision-item-tracking

usage: `python3 item_tracking.py videofile box1 box2 box3 ...`

example: `python3 item_tracking.py Video_sample_1.mp4 "537,50,40,172" "244,74,50,60" "353,310,215,40"`

#### file structure:
- item_tracking.py
- Video_sample_1.mp4
